Day 8: Dictionaries and Maps


HackerRank Python code

Today, we're learning about Key-Value pair mappings using a Map or Dictionary data structure.

×

Copy and Paste below code in python IDLE

import sys

n = int(sys.stdin.readline().strip())
phoneBook = dict()

for i in range(n):
entry = sys.stdin.readline().strip().split(' ')
phoneBook[entry[0]] = entry[1]

query = sys.stdin.readline().strip()
while query:
phoneNumber = phoneBook.get(query)
if phoneNumber:
print(query + '=' + phoneNumber)
else:
print('Not found')
query = sys.stdin.readline().strip()
